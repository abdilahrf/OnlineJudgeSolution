Konversi String

Problem Code : CONVERT
Time Limit : 5 seconds
Memory Limit : 64 megabytes

Kamu diberikan 2 buah string A dan B dengan panjang yang sama dan tidak lebih dari 1000. Tugas kamu adalah mengkonversikan string A menjadi string B.

Setiap karakter pada string A bisa kamu ubah ke karakter setelahnya atau ke karakter sebelumnya. Karakter ini bersifat siklik, yaitu karakter yang berlaku adalah '...XYZABCDEFGHIJKLMNOPQRSTUVWXYZABC...'. Ini berarti sebelum 'A' adalah 'Z' dan setelah 'Z' adalah 'A'. Namun setiap merubah satu karakter, biaya yang dibutuhkan adalah 1 per operasi. Contohnya : untuk mengubah 'A' menjadi 'C' dibutuhkan biaya sebanyak 2 jika kita merubah 'A' ke setelahnya terus. Namun jika kita merubah 'A' menjadi 'C' tapi merubah 'A' dengan cara mengurangi terus menerus, maka membutuhkan biaya sebanyak 24.

Kamu disuruh menentukan berapa cost minimum untuk mengkonversikan string A menjadi string B. Tapi ingat, pilihan huruf mana yang mau ubah pertama, kedua, ketiga, keempat, dan seterusnya berpengaruh terhadap cost tersebut. Misalkan, A = "ABC" dan B = "ZCA". Untuk setiap i (1 ≤ i ≤ panjang karakter A) cost masing masing huruf ke i adalah {25, 1, 2}. Jika kamu memilih merubah huruf kedua dulu, lalu huruf pertama, kemudian huruf terakhir, maka total costnya adalah : (1 * 1) + (2 * 25) + (3 * 2) = 57 (Perhatikan bahwa ini bukanlah cost minimum untuk kasus tersebut).

Input

Baris pertama berisi sebuah bilangan T (T ≤ 100) yang menyatakan banyaknya kasus yang harus ditangani. Untuk setiap kasus berisi 2 baris, baris pertama berisi string A dan baris kedua berisi string B (1 ≤ |A|, |B| ≤ 1000, |A| = |B|, Hanya terdiri dari huruf besar).

Output

Untuk setiap kasus, cetak "Kasus K: M" dimana K adalah nomor kasus dimulai dari 1 dan M adalah total cost terkecil yang mungkin untuk mengkonversikan string A menjadi string B.



Contoh input Output untuk contoh input 
2
ABC
ZCA
JOLLYBEE
JOLLYBEE

Kasus 1: 7
Kasus 2: 0
