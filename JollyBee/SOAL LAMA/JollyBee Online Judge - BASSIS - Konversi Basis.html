Konversi Basis

Problem Code : BASSIS
Time Limit : 3 seconds
Memory Limit : 64 megabytes

Huchiat senang bermain dengan angka, dan dijuluki "pengkonversi basis berjalan". Suatu hari, karena penasaran, Macat dan Sancat menantang Huchiat dalam sebuah permainan. Dalam permainan itu, Macat memberikan sebuah bilangan desimal N dan Sancat memberikan sebuah bilangan B, di mana N dan B adalah bilangan desimal dan basis secara berturut-turut, 1<=N<=2147483647 dan 2<=B<=36. Jika Huchiat bisa menjawab dengan benar semua pertanyaan Macat dan Sancat dengan bilangan N dalam basis B, Macat dan Sancat akan mentraktir Huchiat makan Panda Tei, dan sebaliknya bila Huchiat salah menjawab. Oleh karena itu, bantulah Huchiat menyelesaikan tantangan Macat dan Sancat.

Input

Baris pertama terdiri dari 1 buah bilangan T, banyaknya kasus. T baris berikutnya berisi bilangan N dan B seperti dijelaskan di soal.

Output

Untuk baris i, 1<=i<=T, tuliskan "Case #i: " tanpa tanda petik dan bilangan N dalam basis B.



Contoh input Output untuk contoh input 
2
15 2
234 16
 
Case #1: 1111
Case #2: EA
