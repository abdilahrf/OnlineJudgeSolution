Jumlah dan Selisih

Problem Code : JUMSEL
Time Limit : 3 seconds
Memory Limit : 32 megabytes

Ricat suka sekali bermain dengan angka. Pada suatu hari ia melihat papan tulis di kelasnya penuh dengan angka. Ricat ingin mencari dua buah bilangan bulat positif A dan B sedemikian sehingga (A + B) dan (A - B) terdapat di papan tulis dan (A * B) semaksimal mungkin. A, B, maupun (A * B) tidak harus ada di dalam papan tulis.

Input

Input dimulai dengan sebuah bilangan bulat T (T ≤ 150) yang menyatakan banyaknya kasus yang harus anda tangani.

Setiap kasus dimulai dengan sebuah bilangan bulat N (2 ≤ N ≤ 50) jumlah angka yang ada di papan tulis. Baris berikutnya berisi N buah bilangan Hi (1 ≤ Hi ≤ 100) yang menyatakan angka-angka yang ada di papan tulis.

Output

Untuk setiap kasus, output "Case #X: C" (tanpa tanda kutip) di mana X adalah nomor kasus dimulai dari 1, C adalah hasil perkalian dua bilangan bulat A dan B di mana (A + B) dan (A - B) terdapat di papan tulis dan (A * B) semaksimal mungkin. Asumsikan bahwa pasti terdapat bilangan bulat positif A dan B yang demikian dari input yang diberikan.



Contoh input Output untuk contoh input 
4
3
1 3 5
4
1 4 5 8
2
2 100
5
50 58 47 57 40
 
Case #1: 6
Case #2: 12
Case #3: 2499
Case #4: 441


Penjelasan untuk contoh input 1

Jika A = 3 dan B = 2, maka (A + B) = 5 dan (A - B) = 1 yang keduanya ada di papan tulis. (A * B) adalah 6, dan ini adalah (A * B) terbesar yang bisa anda dapatkan dari angka-angka yang ada di papan tulis.
