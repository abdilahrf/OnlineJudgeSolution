Binary Tree

Problem Code : BINTREE
Time Limit : 3 seconds
Memory Limit : 64 megabytes

Diberikan sebuah struktur data sederhana bernama Binary Tree, dimana setiap node memiliki tepat 2 anak (kiri 1 dan kanan 1). Masing masing node memiliki nomer dari 1 (Yang paling atas atau yang biasa kita sebut root) dimana anak kiri memiliki nomer node 2 * N, dan anak kanan memiliki nomer node (2 * N) + 1, dimana N adalah nomer dari parentnya.

Tugas kalian disini adalah, diberikan N node, dimana tiap nomer node tersebut berbeda beda. Tentukan node terbesar yang merupakan orang tua dari semua N buah node tersebut.

Input

Baris pertama berisi bilangan T (T ≤ 1000) yang menyatakan banyaknya kasus yang harus ditangani. Untuk setiap kasus dimulai dengan sebuah bilangan N (1 ≤ N ≤ 10) yang merupakan banyak node yang akan diberikan. Lalu baris berikutnya berisi N buah bilangan A[i] (1 ≤ A[i] ≤ 250) yang menyatakan nomer node ke-i. Tidak ada node ke-i dan node ke-j dimana i ≠ j yang memiliki nomer node yang sama.

Output

Untuk setiap kasus, cetak "Kasus K: P" dimana K adalah nomer kasus dimulai dari 1 dan P adalah node terbesar yang merupakan orang tua dari N buah node tersebut.



Contoh input Output untuk contoh input 
2

2

4 5

6

8 4 3 10 12 15


Kasus 1: 2

Kasus 2: 1

Hint : Gunakan tipe data long long. %lld atau cin / cout.
