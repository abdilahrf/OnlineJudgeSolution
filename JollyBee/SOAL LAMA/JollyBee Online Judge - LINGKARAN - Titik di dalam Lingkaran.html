Titik di dalam Lingkaran

Problem Code : LINGKARAN
Time Limit : 1 seconds
Memory Limit : 64 megabytes

Diberikan sebuah lingkaran dan sebuah titik. tentukan apakah titik tersebut terletak di garis, di dalam, atau di luar dari sebuah lingkaran


Input

Baris pertama berisi sebuah bilangan bulat T (T ≤ 1000) yang menyatakan jumlah kasus. Setiap kasus dimulai dengan tiga buah bilangan bulat X,Y,R (0 ≤ X,Y ≤ 10000; 1 ≤ R ≤ 10000) yang menyatakan titik pusat (X,Y) dan jari-jari (R). kemudian diikuti oleh sebuah baris yang berisi dua buah bilangan bulat X1,Y1 (1 ≤ X,Y ≤ 10000) yang menyatkan letak titik tersebut

Output

Untuk setiap kasus, output dalam satu baris "Kasus #A: DALAM" jika titik berada di dalam lingkaran, "Kasus #A: LUAR" jika titik berada di luar lingkaran, atau "Kasus #A: GARIS" jika titik berada di garis lingkaran, dengan A adalah nomor kasus dimulai dari 1 secara berurutan.



Contoh inputOutput untuk contoh input
3
0 0 5
3 4
0 0 5
3 3
0 0 5
3 6

Kasus #1: GARIS
Kasus #2: DALAM
Kasus #3: LUAR
