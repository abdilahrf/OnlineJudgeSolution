Huruf Vokal

Problem Code : VOKAL
Time Limit : 1 seconds
Memory Limit : 32 megabytes

Monce sedang belajar mengenal huruf. Monce sangat kesulitan membedakan huruf konsonan dan huruf vokal. Melihat permasalahan ini, ochien (teman-nya monce) ingin membantu monce dalam belajar mengenal huruf. Ochien memberikan soal latihan berupa kalimat yang hanya mengandung karakter A-Z, a-z dan spasi sepanjang N karakter, dimana 0<=N<=100000, tugas monce adalah menghitung jumlah huruf vokal yang terdapat pada kalimat yang diberikan. Oleh karena itu, bantulah monce menyelesaikan soal yang diberikan oleh ochien.

Input

Baris pertama terdiri dari 1 buah bilangan T (1<=T<= 1000), banyaknya kasus. T baris berikutnya berisi kalimat sepanjang N seperti dijelaskan di soal.

Output

Untuk baris i, 1<=i<=T, tuliskan "Kasus #i: " tanpa tanda petik dan bilangan N dalam basis B.



Contoh input Output untuk contoh input 
2
Monce seDang BeLajar Huruf
OcHiEn MenGajar MONCE belajar HuRuF
 
Kasus #1: 9
Kasus #2: 13

Huruf vokal adalah A, I, U, E, O, a, i, u, e, o

Hint : tambahkan getchar() (jika menggunakan c/c++) setelah membaca input jumlah kasus, digunakan untuk membaca karakter new line.
