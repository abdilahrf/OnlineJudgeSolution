Bilangan Dahsyat

Problem Code : DAHSYAT
Time Limit : 3 seconds
Memory Limit : 32 megabytes

Sebuah bilangan bulat dapat dinyatakan keren jika bilangan tersebut bisa di representasikan dalam bentuk K*(K+1)/2, dimana K merupakan sebuah bilangan bulat. Sebuah bilangan bulat juga bisa dinyatakan DAHSYAT jika bilangan tersebut bisa didapat dari penjumlahan dua bilangan keren (tidak harus merupakan dua bilangan yang berbeda).

Diberikan sebuah bilangan N, tentukan apakah N adalah bilangan yang DAHSYAT atau bukan.

Input

Baris pertama berisi sebuah bilangan bulat T (T ≤ 100) yang menyatakan jumlah kasus. Setiap kasus dimulai dengan sebuah bilangan bulat N (1 ≤ N ≤ 109) yang menyatakan sebuah bilangan bulat yang ingin diketahui, apakah N merupakan angka yang DAHSYAT atau tidak.

Output

Untuk setiap kasus, output "Case #X: DAHSYAT" (tanpa tanda kutip), jika N adalah bilangan yang DAHSYAT, atau output "Case #X: TIDAK DAHSYAT" jika N bukan bilangan yang DAHSYAT. X merupakan nomor kasus dimulai dari 1.



Contoh inputOutput untuk contoh input
2
256
512 

Case #1: DAHSYAT
Case #2: TIDAK DAHSYAT
