Penggabungan String

Problem Code : GBNGSTR
Time Limit : 2 seconds
Memory Limit : 64 megabytes

Diberikan 2 buah string masing masing A dan B yang panjangnya tidak melebihi 200. Masing masing string harus diperlakukan sesuai dengan tipenya masing masing. String A dapat dipecah menjadi 3 buah bagian (String kosong juga merupakan bagian). Misalkan "ALGO" dapat dipecah menjadi {"ALGO", "", ""}, atau dapat menjadi {"AL", "GO", ""} atau {"A", "LG", "O"}, atau bisa menjadi {"", "", "ALGO"}, dan seterusnya. Sedangkan string B tidak dapat dipecah dan harus tetap menjadi satu kesatuan utuh. Ingat string A hanya boleh dipotong dan bukan diacak.

Kita ingin menggabungkan string A dengan string B dengan ciri cirinya masing masing. Kita bisa membagi string A menjadi 3 buah bagian, dan bagian yang tengah (bagian kedua) dapat kita hilangkan jika bagian tersebut adalah subsequence dari string B.

Subsequence adalah sebuah string X yang bisa didapat dari string Z jika kita menghapus beberapa bagian dari Z dan tidak boleh mengubah susunan. Contoh X = "ACE" merupakan subsequence dari Z = "ABCDE" karena kita bisa menghapus "B" dan "D" dari string Z tanpa mengubah susunan karakter pada string Z.

Misalkan string A = "BONGKAR" dan string B = "COUNTING"


Hasil akhirnya adalah "BCOUNTINGKAR" dimana panjangnya adalah 12 karakter. Namun kita juga bisa memecah string A dengan cara lain seperti gambar kedua. Untuk cara kedua, hasil penggabungannya adalah "BONCOUNTINGKAR" dengan panjang 14 karakter. Untuk masalah ini kita ingin agar string hasil penggabungan A dan B panjangnya sekecil mungkin.

Input

Baris pertama berisi T (T ≤ 100) yang menyatakan banyaknya kasus yang harus ditangani. Untuk setiap kasus dimulai dengan dua buah bilangan a dan b (1 ≤ a, b ≤ 100) yang menyatakan panjang string A dan panjang string B. Baris berikutnya berisi string A, lalu baris berikutnya berisi string B. Setiap string hanya terdiri dari 'A'..'Z'.

Output

Untuk setiap kasus, cetak "Kasus Y: A" dimana Y adalah nomor kasus dimulai dari 1 dan A adalah panjang karakter terkecil jika string A digabungkan dengan string B.

Contoh input Output untuk contoh input 
3
7 8
BONGKAR
COUNTING
9 9
ALGORITMA
ALGORITMA
5 5
ABCDE
FGHIJ

Kasus 1: 12
Kasus 2: 9
Kasus 3: 10

Penjelasan contoh kasus :
  • Input 1 merupakan ilustrasi contoh pada soal.
  • Kita bisa memecah string A menjadi 3 bagian seperti ini {"", "ALGORITMA", ""} sehingga kita bisa menghilangkan bagian tengah karena "ALGORITMA" merupakan subsequence dari "ALGORITMA" pada string B, sehingga waktu penggabungan akan tetap menjadi "ALGORITMA".
  • Kita bisa memecah string A menjadi {"", "", "ABCDE"}, {"ABCDE", "", ""}, {"ABC", "", "DE"} atau yang lainnya selama bagian tengah kosong. Kita tidak bisa mengisi bagian tengah karena tidak ada kemungkinan bagian dari A yang merupakan subsequence dari string B. Sehingga bagaimanapun hasilnya tetap 10 karena tidak ada string yang hilang.
